BARS

/b…ërz/ | noun. | Biological Assay Record System

Intro

BARS is an open-source and full-stack tool for workflow management with single-cycle drug susceptibility assays. This system will manage plasmid clones, virus stocks, drugs, and plate readings (from the Victor plate reader).

How to get started

  1. The Manage page is for creating new virus stocks and/or new drugs that have been tested. You will be able to reference existing or add new clones.
  2. After getting absorbance values and converting the resulting file into CSV format, use the Enter Assay page to create a new plate reading in the system. Once submitted, the website will automatically redirect to the analysis page for that particular reading.
  3. The Analysis page will generate a dose response curve (with EC50) and % control transformed values (for use in Prism).
  4. The Overview page is for comparing different quandrants.

About

Built using a Flask web server (Python 3.x) with a sqlite3 database. The front end utilizes Angular 1.x for data management and API requests to Flask, D3 for data visualization, and numpy + scipy libraries in Python for sigmoidal regression analysis. Please visit the GitHub page for the complete source code for this project.
